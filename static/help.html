Manual de Ayuda - Funciones BBG

BBGDATAPOINT(isin, field)

Obtiene el valor puntual de un campo en Bloomberg.

Parámetros:

  • isin: El ISIN del instrumento.
  • field: El mnemónico del campo Bloomberg.

Retorno: El valor del campo.

BDP(isin, field)

Obtiene el valor puntual de un campo en Bloomberg.

Parámetros:

  • isin: El ISIN del instrumento.
  • field: El mnemónico del campo Bloomberg.

Retorno: El valor del campo.

BBGFIELDNAME(field)

Obtiene el nombre de un campo Bloomberg.

Parámetros:

  • field: El mnemónico del campo Bloomberg.

Retorno: El nombre del campo Bloomberg.

BBGDATA(isin)

Obtiene todos los campos de Bloomberg de un instrumento.

Parámetros:

  • isin: El ISIN del instrumento.

Retorno: La lista de campos y sus valores.

BBGFIELD(field)

Obtiene la descripción de un campo de Bloomberg.

Parámetros:

  • field: El mnemónico del campo Bloomberg.

Retorno: Una lista de valores con la descripción del campo de Bloomberg.

BBGCASHFLOW(isin)

Obtiene el cash flow de un instrumento.

Parámetros:

  • isin: El ISIN del instrumento.

Retorno: Una lista de tres columnas con DES_CASH_FLOW_DT (fecha), DES_CASH_FLOW_CPN_AMT (cupón de amortización), DES_CASH_FLOW_PRINC_AMT (principal).

BBGFACTORSCHEDULE(isin)

Obtiene el calendario de factores de un instrumento.

Parámetros:

  • isin: El ISIN del instrumento.

Retorno: Una lista de tres columnas con BC_FACTR_SCHEDULE_DT (fecha), BC_FACTOR_SCHEDULE_FACTOR (factor).

BBGHISTORY(isin, fields, start_date, end_date)

Obtiene una lista de campos históricos entre dos fechas.

Parámetros:

  • isin: El ISIN del instrumento.
  • fields: La lista de mnemónicos de campos Bloomberg (separada por coma, ej: 'DATE,PX_LAST').
  • start_date: La fecha de inicio de la consulta.
  • end_date: La fecha de fin de la consulta.

Retorno: Una lista con una columna por cada campo solicitado, para todas las fechas del rango.

BDH(isin, fields, start_date, end_date)

Obtiene una lista de campos históricos entre dos fechas.

Parámetros:

  • isin: El ISIN del instrumento.
  • fields: La lista de mnemónicos de campos Bloomberg (separada por coma, ej: 'DATE,PX_LAST').
  • start_date: La fecha de inicio de la consulta.
  • end_date: La fecha de fin de la consulta.

Retorno: Una lista con una columna por cada campo solicitado, para todas las fechas del rango.

BBGFIELDS(filter)

Obtiene una lista de campos Bloomberg que coinciden con el filtro.

Parámetros:

  • filter: El filtro de búsqueda.

Retorno: Una lista de campos Bloomberg con varias columnas.
